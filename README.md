Sistem Penyimpanan dan Pengelolaan Aspirasi Warga dalam sebuah RT/RW ini merupakan proyek ahkir dari kelompok 6.
Anggota Kelompok:
1. Ahmad Rovii (2500269)
2. Alfons Ikhsan Dwi Pamungkas (2512339)
3. Muhammad Ikhsan Akbar (2503014)
4. Rapida P. Simanjuntak (2503005)
5. Resthya Raedhina Puteri (2504317)
6. Ridwan Akmal Suandi (2504828)
7. Sulthon Al Waafi (2502413)

Proyek ini memiliki tujuan sebagai berikut:
1. Merancang aplikasi penyimpanan dan pengelolaan aspirasi warga dalam sebuah RT/RW.
2. Memudahkan Ketua RT/RW dalam menerima dan menyimpan seluruh aspirasi Masyarakat.
3. Membantu Masyarakat menyampaikan aspirasi nya tanpa terkendala.
4. Menjadikan sistem pada sebuah RT/RW terarah.
5. Menjadikan aplikasi penyimpanan dan pengelolaan aspirasi warga lebh efektif dan berkesinambungan dan lebih efektif dalam sebuah RT/RW.
6. Membantu penyusunan laporan yang lebih optimal.

Fitur Utama:
1. Tambah aspirasi warga
2. Lihat semua aspirasi
3. Cari aspirasi berdasarkan nama
4. Mengurutkan aspirasi sesuai abjad
5. Hapus aspirasi berdasarkan ID

Konsep Pemrograman yang Digunakan:
1. Struktur Data: struct dan array of struct
2. User Interaction
3. Presistensi Data
4. Modular Programming
5. Searching: Sequential Search
6. Sorting: Bubble Sort
